# Plan 索引

> 计划文件统一放在这里。用有意义的名字，不要随机动物名。
> 执行完标记状态，废弃的移到 `archive/` 或删除。

## 约定

- **文件名**：`<功能>-<简述>.md`，一眼能看出是干什么的
- **状态**：`📋 计划中` / `🚧 执行中` / `✅ 完成` / `❌ 废弃`
- **完成后**：更新状态 + 在 [iteration-log.md](../iteration-log.md) 写一笔关键决策
- **不要攒尸**：废弃的计划直接删，完成了的超过一个月可归档
